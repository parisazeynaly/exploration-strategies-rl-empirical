## Project Overview

This project empirically compares three exploration strategies within the Q-learning framework across multiple environments of varying complexity and stochasticity.

**Research Questions:**
- **RQ1:** Do Bayesian and classical exploration strategies differ significantly in performance and convergence speed?
- **RQ2:** Does Bayesian exploration provide greater robustness under environmental uncertainty?

**Three strategies compared:**

| Strategy | Type | Mechanism |
|---|---|---|
| Epsilon-Greedy | Classical | Random with prob ε, greedy otherwise |
| Softmax (Boltzmann) | Classical | Probability proportional to exp(Q/τ) |
| Thompson Sampling | Bayesian | Add uncertainty noise scaled by visit counts |

## Methods

### Q-learning Update (shared by all strategies)
```
Q(s,a) ← Q(s,a) + α · [r + γ · max Q(s') − Q(s,a)]
```

### Action Selection

**Epsilon-Greedy:**
```
if rand() < ε  →  random action
else           →  argmax Q(s,a)
```

**Softmax:**
```
P(a) = exp(Q(s,a) / τ) / Σ exp(Q(s,a') / τ)
```

**Thompson Sampling:**
```
uncertainty    = temperature / (counts[s,a] + 1)
sampled_values = Q(s,a) + N(0,1) × uncertainty
action         = argmax(sampled_values)
```

### Key Metric: Total Reward (Area Under Learning Curve)
```python
total_reward = np.sum(rewards)   # sum of all rewards across all episodes
```
Captures both learning speed and final performance — strategies that converge earlier earn more total reward.

## Key Results

| Condition | Best Strategy | Worst Strategy |
|---|---|---|
| 4×4 Deterministic | Thompson (fastest convergence) | Epsilon-Greedy (never converges) |
| 8×8 Deterministic | Thompson (converges ~1800 ep) | Epsilon-Greedy (reward=0) |
| 4×4 Stochastic | Thompson (most robust) | Softmax (collapses) |
| 8×8 Stochastic | Thompson (best) | Softmax (near-zero) |
| CliffWalking | Thompson | Epsilon-Greedy |
| Taxi | Thompson | Epsilon-Greedy |

**Sensitivity:** Softmax is the most sensitive to parameter choice (collapses at τ > 0.3). Thompson degrades gracefully. Epsilon-Greedy is insensitive but performs worst overall.

## Statistical Tests

- **Kruskal-Wallis** — non-parametric test across all 3 strategies (no normality assumption)
- **Mann-Whitney U** — pairwise comparisons with Bonferroni correction (α = 0.05/3)
- **Cohen's d** — effect size between Thompson and each classical strategy
